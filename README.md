## Requirement
* R-studio with R >= 4.3.0
* Packages
  * Data manipulation: tidyverse, data.table, lubridate, scales, zoo, janitor
  * Spatial data: sf, raster
  * Plots: RcolorBrewer, ggspatial, ggrepel, ggtext, ggdist, grid, lemon, cowplot, patchwork
